PROJETO
-------------------

API para cadastro de produtos e listagem de categorias.


REQUERIDO 
------------

PHP 7.4.
MySQL

CONFIGURATION
-------------

### Database

Edit the file `config/db.php` with real data, for example:

```php
return [
    'class' => 'yii\db\Connection',
    'dsn' => 'mysql:host=localhost;dbname=yii2basic',
    'username' => 'root',
    'password' => '1234',
    'charset' => 'utf8',
];
```


EXECUCAO DO PROJETO
------------
~~~
composer update  
~~~
~~~
php yii migrate
~~~
~~~
php -S localhost:8080 -t web
~~~
